Kullanım Kılavuzu

Nedir?

WSDL Client Generator, SOAP servis WSDL dosyalarından otomatik olarak Java client projesi üretir. Hem Prod hem Test ortamı destekler; çıktı olarak kaynak kodları, WSDL dosyaları ve kullanıma hazır JAR içeren bir ZIP paketi sunar.

1 · WSDL Kaynağı
PROD Zorunlu
URL girerek veya .wsdl / .xml dosyası yükleyerek belirtin. Her iki yöntem de desteklenir.
TEST Opsiyonel
Girilmezse Prod WSDL kopyalanır. Test ortamı endpoint farklıysa ayrıca belirtilmesi önerilir.
Kod üretimi hangi WSDL'den?
Java stub sınıfları varsayılan olarak Prod WSDL'den üretilir. Prod ile Test şeması farklıysa bunu seçerek değiştirebilirsiniz.
2 · Java Versiyonu
Java 8 / 11
  • JAX-WS RI kullanır
  • javax.xml.ws.* namespace
  • javax.xml.bind.* (JAXB)
  • Legacy projeler için uygun
Java 17 / 21 Önerilen
  • Apache CXF 4.x kullanır
  • jakarta.xml.ws.* namespace
  • jakarta.xml.bind.* (JAXB)
  • Modern Spring Boot ile uyumlu
3 · ZIP İçeriği
📄 pom.xml
Maven proje tanımı. Bağımlılıklar ve yeniden üretim plugin'i dahildir.
📁 src/main/resources/wsdl/
Prod ve Test WSDL dosyaları. ServiceBean bu dosyaları classpath'ten okur.
📁 src/main/java/…/
Üretilen stub sınıfları ve util/XxxServiceBean.java.
📦 lib/*.jar
Tüm bağımlılıkları içeren fat JAR. Maven'a install-local-m2.bat ile kurulur.
⚙️ install-local-m2.bat / .sh
JAR'ı yerel .m2 deposuna kurar. Diğer projelerde bağımlılık olarak kullanmak için çalıştırın.
4 · Projeyi Kullanma
1
ZIP'i Aç & IntelliJ'e Aktar
ZIP'i çıkartın, IntelliJ'de File → Open ile pom.xml seçin. Maven projesi olarak tanınır.
2
Yerel Depoya Kur
install-local-m2.bat'ı çalıştırın. Bu, JAR'ı ~/.m2'ye kopyalar; başka Maven projelerinden <dependency> ile eklenebilir hale gelir.
3
ServiceBean ile Servis Çağrısı
Üretilen XxxServiceBean.SINGLETON.getPort() üzerinden metotları çağırın. Ortam, kullanıcı adı ve şifre SOA_ENVIRONMENT, SOA_WEBSERVICE_USERNAME, SOA_WEBSERVICE_PASSWORD ortam değişkenlerinden okunur.
4
WSDL Değişince Yeniden Üret
src/main/resources/wsdl/ altındaki WSDL'i güncelleyin, ardından mvn clean install çalıştırın. Plugin stub sınıflarını otomatik olarak yeniden üretir.
Örnek Kullanım
// ServiceBean.SINGLETON otomatik ortam seçimi yapar
MuhasebePortType port = MuhasebeServiceBean.SINGLETON.getPort();
GetMuhasebeFisResponse resp = port.getMuhasebeFis(request);
Bağımlılık Olarak Eklemek İçin
<dependency>
    <groupId>com.example</groupId>
    <artifactId>soap-client</artifactId>
    <version>1.0.0</version>
</dependency>
